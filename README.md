## 🛠️ Cambios realizadas

- **Integracion de Mongo Atlas:** El almacenamiento local le migre a una base de datos en la nube paro no instalar app pero garantizando que los datos no se pierdan si se reinicia el servidor.
- **Seguimiento de usuarios:** El codigo cuenta con la implementacion de express-session que muestra los usuarios conectados al chat de soporte.
- **Mediador:** Se encarga de bloquear el acceso a las rutas para agregar, eliminar y editar los productos si el usuario no inicio sesion.
- **Chat de soporte en vivo:** Es un chat en tiempo real usando Socket.io.
- **Arquitectura MVC:** El codigo esta organizado en Vistas, Modelos, Rutas y Controladores.
- **Sincronizacion:** Los mensajen se pueden ver en tiempo real en varias pestañas sin refrescar la pagina.
- **Layouts:** El diseño fue unificado en el archivo principal.hbs para que la barra de navegacion sea la misma.
